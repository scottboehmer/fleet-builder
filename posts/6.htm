Copy Markdown

28 October 2025

The fleet builder has had the ability to show a markdown version of a fleet for a while now. That was intended to make it easy to copy-paste it to anywhere you wanted with formatting that would work well in apps like Discord.

I've taken that a step further now. There is now a button that will add the markdown version of your fleet directly to your clipboard without having to switch to the markdown view or select text. My hope is that this will be easier to use, especially on mobile devices.

A fan project built by Scott Boehmer for the Leviathans: The Great War game by Catalyst Game Labs
